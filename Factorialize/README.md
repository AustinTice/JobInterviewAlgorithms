# Factorialize
**Problem Definition:** 
Write a function that takes a number as an argument and return its factorial value.

---
**Examples:**
* `Factorialize(5) = 5 * 4 * 3 * 2 * 1 = 120`
* `Factorialize(3) = 3 * 2 * 1 = 6`

---
**Notes:**
* A common way of solving this problem is through recursion.
